# Mo小窝 (Momo's Small Site) - 项目详细文档

## 1. 项目概述

**Mo小窝** 是一个由 **Momo同学** 开发的个人创意前端展示网站。该项目不仅是作者展示前端技术能力的个人主页，也是一个集成了多种趣味小工具、经典游戏和生活记录的综合性互动平台。

- **项目定位**：个人技术作品集、生活感悟分享、前端实验场。
- **核心理念**：人海相逢，自有缘分；人生是旷野，自由如风。

## 2. 技术栈与架构

项目主要基于静态网页技术构建，融合了多种现代前端库和动画特效。

### 核心技术
- **基础框架**：HTML5, CSS3, JavaScript (原生 & jQuery)
- **动画库**：
    - [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)：用于高性能、复杂的页面补间动画。
    - [Animate.css](https://animate.style/)：提供预设的 CSS 动画效果。
- **视觉特效**：
    - [Three.js](https://threejs.org/)：用于实现 3D 效果（如 3D 图片墙）。
    - [Particles.js](https://vincentgarreau.com/particles.js/)：用于背景粒子散点动画。
- **组件/插件**：
    - [Swiper](https://swiperjs.com/)：用于实现移动端兼容的触摸滑动效果。
    - [TypeIt](https://typeitjs.com/)：用于打字机文字特效。
    - [FontAwesome](https://fontawesome.com/)：图标库。

### 部署与工具
- **部署平台**：[Vercel](https://vercel.com/)
- **联系表单**：集成 [FormSubmit](https://formsubmit.co/) 提供无需后端代码的邮件反馈功能。

## 3. 核心功能模块

### 3.1 首页 (Landing Page)
- **交互体验**：具有视差滚动效果、平滑滚动动画。
- **背景音乐**：支持全局背景音乐播放及用户交互后的自动恢复。
- **入口设计**：通过“进入”按钮引导用户进入核心功能区。

### 3.2 登录系统
- **访问控制**：提供简单的用户名密码验证（用户名: `Momo`, 密码: `5201314`）或“游客访问”模式。
- **运行统计**：实时展示网站自 2025 年 6 月 30 日起的累计运行时间。

### 3.3 内部主站 (Dashboard)
- **首页**：包含粒子特效背景、随机语录切换（涵盖励志、幽默、感悟等多维度内容）。
- **关于我**：展示作者的技能图谱（HTML/CSS/JS/Vue/React, Python/Java, 数据库, 视频剪辑等）及个人介绍。
- **我的博客**：分为科技日常、思绪日常、行走日常、阅读日常四个专区。
- **联系我**：提供反馈表单和作者的社交媒体（抖音、YouTube、X）链接。

### 3.4 互动 Demo 库 (Projects)
包含多个独立的前端互动项目：
1.  **手势圣诞🌳**：通过手势互动的圣诞树装饰。
2.  **花式图片册**：基于 Three.js 的 3D 旋转图片墙（球体、螺旋、立方体等模式）。
3.  **爱心纪念册**：浪漫主题的图片与情感展示页面。
4.  **穿越时空的旅行**：基于 Swiper 的幻灯片展示，记录旅行瞬间。
5.  **视频解析播放器**：提供第三方视频解析功能的播放器。
6.  **俄罗斯方块**：经典的休闲益智游戏。
7.  **经典五子棋**：支持人人对弈的棋类游戏。
8.  **弹窗生成器**：自定义文字生成的趣味弹窗互动。
9.  **代码里的温暖心意**：教学式的爱心代码绘制。
10. **逛逛百度**：百度首页风格的创意纪念页面。
11. **搞笑漫画**：趣味性的内容分享。

## 4. 目录结构说明

- `/login_in/`：项目的核心功能区，包含登录页及内部主页。
    - `/game/`：存储各种互动游戏项目。
    - `/css/` & `/js/`：主站点的样式与脚本逻辑。
    - `/link/`：包含音乐资源、自荐页面等。
    - `/lovestory/`：爱心纪念册相关资源。
    - `/travel_plane/`：旅行记录相关资源。
- `/wp-content/`：复用了 WordPress 主题（CreatorsPC）的静态资源，用于首页的 UI 框架。
- `/images/`：存放全站通用的图像资源。
- `index.html`：项目根目录的落地页。

## 5. 开发与维护

- **推荐环境**：建议使用最新版 Chrome、Edge 或 Firefox 浏览器以获得最佳动画体验。
- **贡献方式**：支持通过 GitHub 提交 Issue 或 Pull Request 参与贡献。
